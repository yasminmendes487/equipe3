## 🛠️ Como contribuir (mini-lab)
```bash
# 1. Clone o repositório
git clone https://github.com/yasminmendes487/equipe3.git
cd equipe3

# 2. Crie sua branch
git checkout -b setup/readme-seunome

# 3. Faça suas alterações e commit
git add .
git commit -m "Adiciona integrante Seunome ao README"

# 4. Envie para o GitHub e abra um PR
git push origin setup/readme-seunome
```
Depois acesse o repositório no GitHub e clique em **"Compare & pull request"**.
